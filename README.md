# python-basics-to-advanced
A structured collection of Jupyter Notebooks documenting my journey from Python fundamentals to advanced library applications.

---

## 🛠️ Conda Environment Cheat Sheet
Essential commands used to manage the environments for these projects.

| Task | Command |
| :--- | :--- |
| **Create Env** | `conda create -n python_study python=3.10` |
| **Activate** | `conda activate python_study` |
| **Install Libs** | `conda install pandas numpy matplotlib` |
| **List Libs** | `conda list` |
| **Deactivate** | `conda deactivate` |

## 📂 Curriculum

### 🟢 Phase 1: Basics of Python
*Foundational concepts and syntax.*
* **Folder:** `01_Basics_of_Python`
* **Content:** Variable types, String manipulation, Integers, and Basic I/O.

### 🟡 Phase 2: Intermediate Python
*Focus on data structures and logic.*
* **Folder:** `02_Intermediate_Python`
* **Content:** Advanced Variable usage, Lists, Dictionaries, and Logic Flow.

### 🔵 Phase 3: Python Libraries
*Data science and utility libraries.*
* **Folder:** `03_Libraries`
* **Content:** Deep dives into NumPy for math and Pandas for data manipulation.
